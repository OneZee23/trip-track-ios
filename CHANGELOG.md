# Changelog

Все заметные изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [0.1.1] -- 7 апреля 2026

### Исправлено
- Убрана демо-поездка при первом запуске — чистый старт без фейковых данных. Существующие демо-поездки удаляются автоматически
- Смягчён алгоритм удаления GPS-аномалий — меньше ложных срабатываний на кривых дорогах и развязках
- Исправлена иконка Telegram в профиле — заменена на чистый paper plane без внешнего круга
- Дефолтная машина переименована из «Телега» в «Ваша машина» / «Your car»
- Переключение между машинами в гараже работает по всей области кнопки
- Карта регионов сразу в правильной теме (тёмная/светлая) без захода на экран записи
- Удаление поездки теперь физически удаляет данные и очищает территории
- Тактильный отклик при остановке записи срабатывает надёжнее (в т.ч. из Live Activity)
- Превью маршрута в ленте теперь всегда соответствует реальному маршруту (исправлена сортировка точек)
- Города в Regions определяются по geocode cache координат, а не по парсингу заголовка поездки

### Изменено
- «Author» → «Developer» в профиле
- Ранг «Novice» переименован в «Beginner»
- Расширены описания в онбординге — упоминание статистики, расхода топлива и гаража
- Версия в профиле читается из Bundle (не хардкод)
- Дефолтная валюта топлива: € вместо ₽
- Улучшен рендеринг тумана на карте регионов — мягче переходы, нет зеленоватого оттенка
- В Regions убрана фиксированная площадь — теперь только исследованные км²
- Проценты исследования в кружках городов и регионов теперь со знаком %
- Убраны стрелки-chevron у городов и регионов (нет перехода)
- «Most explored» вместо вводящего в заблуждение процента от случайного региона
- Оптимизирована производительность маршрута на карте — группировка по скоростным зонам вместо отдельных сегментов

### Добавлено
- Тап по логотипу в ленте — скролл наверх
- Повторный тап по вкладке «Лента» из деталки поездки — возврат назад
- Статистика в профиле (серия, км, поездки) теперь кликабельна — открывает экран статистики
- Быстрая статистика в ленте (trips, km, time) теперь кликабельна — открывает расширенную статистику
- Разрывы GPS на маршруте больше не соединяются прямыми линиями — пустоты вместо артефактов
- Валюта топлива сохраняется per-trip — при смене валюты старые поездки помнят свою
- Pull-to-refresh на экране Regions
- Кеширование данных Regions — мгновенный повторный вход на экран

## [0.1.0] -- 24 марта 2026

Первый релиз. Полнофункциональный автодневник поездок.

### Добавлено

**Запись маршрута**
- GPS-трекинг с фильтром Калмана (`SmoothTrackManager`) для сглаживания координат и удаления шума
- Фоновая запись маршрута (`UIBackgroundModes: location`)
- Реалтайм-показатели во время записи: текущая скорость, высота, пройденная дистанция, время
- Сворачиваемая панель метрик (compact/expanded `TrackingHUD`) -- тап переключает между режимами
- Компактный HUD (`CompactTrackingHUD`) с минимальной информацией
- Спидометр (`SpeedometerView`) с визуальной шкалой
- Индикатор качества GPS-сигнала (`GPSIndicatorView`)
- Плавная анимация "головы" маршрута с glow-эффектом (`GlowingHeadOverlay`)
- Автоматический zoom карты по скорости движения (ближе на малых скоростях, дальше на высоких)
- Пауза и возобновление записи
- Автоматическая темная/светлая тема карты по времени суток (`SunCalculator` -- расчет sunrise/sunset по координатам)
- Idle-экран с пульсирующей кнопкой старта, суммарным пробегом и количеством поездок
- Экран итогов после завершения поездки (`TripCompleteSummaryView`): карта маршрута, статистика, заработанные XP, полученные бейджи, кнопки добавления фото и заметок

**Лента поездок**
- Группировка поездок по месяцам с collapsible-секциями
- Contribution calendar -- GitHub-style heatmap с интенсивностью по километрам за день
- Фильтрация по дате через calendar picker (выбор диапазона)
- Фильтрация по региону (чипы уникальных регионов из геокодинга)
- Поиск по названию поездки
- Быстрая статистика в header: общий пробег, количество поездок, суммарное время
- Пагинация -- 20 поездок на страницу, подгрузка при скролле
- Карточки поездок с превью маршрута (`LightRoutePreview`), названием, датой, дистанцией, временем
- Swipe-удаление с undo-таймером (5 секунд) и toast-уведомлением
- Skeleton-загрузка карточек при первом открытии
- Empty state с предложением начать первую поездку
- Подробная статистика (`StatsView`): текущий/прошлый месяц, недельная активность (Swift Charts), calendar heatmap, топ мест, рекорды

**Детали поездки**
- Интерактивная карта с маршрутом и parallax-stretch при скролле
- Маршрут раскрашен градиентом по скорости (`RouteMapView`)
- Статистика: дистанция, время, средняя/максимальная скорость, набор высоты
- Фотографии поездки -- добавление через `PhotoPickerView`, полноэкранный просмотр (`PhotoFullScreenView`), удаление
- Редактирование названия поездки (inline edit с фокусом)
- Автоматический геокодинг: название формируется из начальной и конечной точек. При ошибке -- retry
- Share-карточка (`TripShareCardView`) для Instagram/соцсетей: маршрут + статистика + брендинг
- Добавление текстовых заметок (`NotesEditorView`)
- Бейджи, заработанные за поездку (`TripBadgesRow`)

**Карта территорий**
- Scratch-карта (fog of war) на основе geohash precision-6 (`GeohashEncoder` + `VisitedGeohashEntity`)
- Fog-маска генерируется на GPU (`FogMaskGenerator`, `FogOverlayRenderer`)
- Fullscreen-режим карты (`FullscreenFogMapView`)
- Статистика: количество посещенных тайлов
- Списки посещенных городов и регионов с количеством визитов
- Все маршруты всех поездок отображаются как polyline overlay
- Backfill при первом запуске -- обрабатывает существующие поездки

**Геймификация**
- 30+ бейджей в 4 категориях: дистанция, исследование, особые, серии
- Бейджи дистанции: Первый путь, Завсегдатай (10 поездок), Дорожный воин (50), километровые вехи (1/100/1000/10000 км), Марафонец (42.2 км за поездку), Стоик (100+ км за поездку), Железная задница (500+ км за поездку), Спид-демон (120+ км/ч)
- Скрытые бейджи (`isHidden`) -- не видны до получения
- Повторяемые бейджи (`isRepeatable`) -- можно получить несколько раз
- Celebration-экран с конфетти при получении бейджа (`BadgeCelebrationView`, `ConfettiView`)
- Детальный overlay для каждого бейджа (`BadgeDetailOverlay`)
- Каталог всех бейджей (`BadgesView`) с группировкой по категориям
- XP-система: 1 XP за километр, x2 бонус за поездки 200+ км, +20 XP за первую поездку дня, +50 XP за новый регион, x1.5 за поездку в новом регионе
- Уровни водителя (1-30) с прогрессивной шкалой XP (`LevelSystem`)
- Ранги: Новичок (1-4), Водитель (5-9), Путешественник (10-14), Исследователь (15-19), Навигатор (20-24), Дальнобойщик (25-29), Легенда (30)
- Daily streak -- серия дней с поездками
- Профиль (`ProfileView`) с выбором аватара (8 эмодзи), карточкой уровня (`DriverLevelView`), быстрой статистикой

**Коллекция дорог**
- Автоматическое обнаружение часто проезжаемых дорог (`RoadCollectionManager`)
- Уровни освоения дороги (discovered, familiar, mastered)
- Карточки дорог с информацией о редкости
- Фильтрация по редкости

**Гараж**
- Профиль автомобиля с одометром (`VehicleCardView`)
- Экран управления машинами (`GarageView`)

**Интерфейс**
- Кастомный glass tab bar (`CustomTabBar`) -- 3 вкладки: лента, запись, территории
- Центральная кнопка записи (оранжевая пульсация в idle, красный стоп при записи)
- UI-модификаторы для единообразия карточек: `.surfaceCard()`, `.glassBackground()`, `.glassPill()`
- Адаптивная цветовая палитра (`AppTheme`) для светлой и темной темы
- Toast-уведомления (`ToastView`)
- Баннер записи (`RecordingBanner`) -- видно на всех вкладках
- Haptic feedback на все интерактивные элементы

**Настройки**
- Тема оформления: системная / светлая / темная (`ThemeManager`)
- Язык интерфейса: русский / английский -- переключается на лету без перезапуска (`LanguageManager`)
- Все строки UI через `AppStrings` enum -- нет hardcoded текста
- Отображение версии и номера сборки из bundle

**Онбординг**
- 3 страницы при первом запуске: приветствие, описание записи, запрос разрешения на геолокацию
- Swipe между страницами (TabView с page style)
- Кнопка "Поехали!" на финальной странице запрашивает location permission

**Данные и persistence**
- CoreData с 8 entities: `TripEntity` (центральная), `TrackPointEntity`, `TripPhotoEntity`, `VehicleEntity`, `UserSettingsEntity`, `VisitedGeohashEntity`, `RoadEntity`
- Cascade-удаление: удаление поездки удаляет все track points и фото
- Batch saves: точки GPS батчатся по 10 штук или каждые 15 секунд (`saveBatchSize`, `saveInterval`)
- Асинхронное сохранение (`PersistenceController.saveAsync()`) для неблокирующих записей из location callbacks
- Binary polyline encoding (`Trip.encodePolyline/decodePolyline`) -- компактное хранение маршрута в Float32 парах
- Превью-полилайн для карточек ленты (упрощенный маршрут)
- Автоудаление мусорных поездок: <500 метров AND <2 минуты
- Очистка orphaned trips (без endDate) при старте
- Фото хранятся в Documents directory (`PhotoStorageService`), миниатюры через `AsyncThumbnailView`

**Разработка**
- Provider pattern для location: `LocationProvider` протокол, `RealGPSProvider` (CoreLocation) + `SimulatedLocationProvider` (dev-джойстик)
- Dev mode с виртуальным джойстиком для тестирования без реального GPS
- Dev menu (`DevMenuView`) для переключения режимов
- Мониторинг сети (`NetworkMonitor`)
- Сборка через `xcodegen` (`project.yml`), локальный signing через `Local.xcconfig` (gitignored)
- Нулевые внешние зависимости -- 100% нативные фреймворки Apple
